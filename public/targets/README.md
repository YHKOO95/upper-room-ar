# Image Targets

`upper-room-targets.mind` 파일을 이 폴더에 넣으면 앱이 4개의 이미지 타겟을 인식합니다.

1. 오브제에 붙일 이미지 4장을 준비합니다.
2. MindAR 이미지 컴파일러로 4장을 하나의 `.mind` 파일로 변환합니다.
3. 생성된 파일명을 `upper-room-targets.mind`로 바꿔 이 폴더에 저장합니다.
4. 이미지 순서는 앱의 `targetIndex`와 연결됩니다.

권장 이미지:

- 선명한 대비가 있고 반복 패턴이 적은 이미지
- 글자만 있는 이미지보다 그림, 질감, 형태가 섞인 이미지
- 너무 반짝이거나 투명한 재질이 아닌 인쇄물
- 실제 현장에서 8cm 이상으로 보이는 크기
